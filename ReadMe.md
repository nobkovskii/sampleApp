# SampleApp

- UI : Thymeleaf + SprinBoot Apps
- API : SprinBoot Apps

# How to use?

git clone.

## boot ui

cd ui  
gradlew.bat build  
gradlew.bat run  
access to `http://localhost:8000`

## boot api

cd api  
gradlew.bat build  
gradlew.bat run

## ui to api

access to `http://localhost:8000/test001`
